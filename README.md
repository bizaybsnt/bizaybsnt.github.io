# Bijay Basnet - Personal Portfolio Website

A modern, responsive personal portfolio website showcasing my professional journey as a Software Engineer. Built with clean HTML, CSS, and JavaScript, featuring bilingual support (English/Japanese) and an elegant design.

## 🌟 Features

- **Bilingual Support**: Seamless language switching between English and Japanese
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Interactive Elements**:
  - Particle.js animated background in hero section
  - Language preference saved in localStorage
- **Professional Sections**:
  - Hero with profile photo
  - About Me
  - Career Journey Timeline
  - Skills & Technologies
  - Social Media Links

## 🚀 Live Demo

Visit the live website: https://bizay.jp/

The website supports:
